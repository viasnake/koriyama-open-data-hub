# 出典・ライセンス・注意事項

この API は、郡山市が公開するオープンデータと公式サイト RSS を利用しています。

## 出典

- [郡山市オープンデータ](https://www.city.koriyama.lg.jp/soshiki/21/176730.html)
- [公共施設等情報](https://www.city.koriyama.lg.jp/soshiki/21/176727.html)
- [防災情報](https://www.city.koriyama.lg.jp/soshiki/21/176726.html)
- [郡山市公式サイト RSS 配信の案内](https://www.city.koriyama.lg.jp/site/userguide/25998.html)
- [新着情報 RSS フィード](https://www.city.koriyama.lg.jp/rss/10/list1.xml)

レスポンスの `meta.source_name` は `郡山市オープンデータ`、`meta.license` は `CC BY 4.0` です。データを利用する場合は、郡山市の公式ページと各データの条件を確認し、必要な出典表示を行ってください。

## 非公式サービス

Civic Koriyama Data は非公式サービスです。郡山市公式サービスではありません。

データの正確性、完全性、最新性、可用性は保証しません。行政手続き、避難、医療、安全確認など、正確性が重要な用途では必ず郡山市公式ウェブサイトまたは関係機関で確認してください。

## 利用時のお願い

- Civic Koriyama Data の旧称は「郡山市オープンデータ非公式 WebAPI」です。
- 過度なリクエストや、短時間に集中する機械的な取得は避けてください。
- キャッシュできるレスポンスはアプリケーション側でキャッシュしてください。
- 元データや出典ページの更新により、レスポンスの内容や件数が変わることがあります。
- 旧 v1 API は提供を終了しています。現在は v2 API を利用してください。
